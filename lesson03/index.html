Notes

The Window Object

  • Using window.alert will display a message in a dialog box. Ex. window.alert('Hello World!')
  • Using window.confirm() method will stop the execution of the program and display confirmation box. Ex. window.confirm('Do you wish to continue?')
  • Using window.prompt() will stop the program and ask for user to input in the dialog box. Ex. window.prompt('Please enter your full name:')
  • You can use window.open(enter url) to have a new window open to that specific url you entered.
  • You use window.close() when you want to close a window.

The Document Object

  • You can use a document.write() to display a string of text on the webpage.
  • You don't really want to use document.write(), it's better to jsut stick with html so you don't mix up javascript with it.

Cookies

  • Cookies are files with information saved that are on user's computers. Cookies are used to keep track of user choices, user preferences, authentication, etc.
  • You assign a cookie by using document.cookie = 'whatever cookie you want to be stored'
  • Applications that contains sensitive information shouldn't rely on cookie expiry.

Timing Functions

  • setTimeout() acepts callback and number of miliseconds for how long of delay before it calls.
  • setInterval() is similar to setTimeout() but this method will continue calling after certain number of milliseconds.
