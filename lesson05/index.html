Ajax

  • Using Ajax is a way to update web pages without reloading the page.
  • This has made web pages no longer static, but instead dynamic.
  • Ajax allows javascript to request information from the server in place of the client.

The Fetch API

  • When fetching an API, you are requesting and sending data across the internet.
  • You use fetch(enter url) to gather data from the resource you wish to use.
  • Using text() method transofrm the text into a JS string.
  • The blobl() method is used to read raw data, such as an image or spreadsheet.
  • One of the most common methods in Ajax is the json() method.
