Forms

  • You can use a form tag that takes input fields, buttons, menus, etc. These are great to use for multiple things, such as registration form, login form, etc.
  • Here goes a simple form where you can type what you want to search then click button to submit.

Input Fields

  • There are multiple kinds of input types you can use. Here are a few.
  • Text Input
  • Password Input (Notice that when user types, the characters are hidden.)
  • Checkbox Input (Can check one or multiple options)
  • Radio Button Input(Only one is selected)
  • There are many kinds of inputs.

Client Side Validation

  • This helps ensure that all forms are filled out and are in correct format.
  • Error messages can pop up if some form wasn't filled out right.
